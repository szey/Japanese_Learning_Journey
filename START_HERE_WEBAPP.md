# Start Here: Web App Direction

This project is now focused on the **web app** direction.

## Current product direction

Mirai Nihongo / Japanese Learning Journey is now being developed as a:

**web-first Japanese learning app for true beginners**

The priority is no longer hardware prototyping.
The e-ink device exploration is treated as archived background thinking, not the current main path.

## What matters now

The active project areas are:

- `webapp/` — Next.js application scaffold
- `backend/` — Go API backend
- `packs/` — stage content packs such as `pack-pre-n5-v1`
- `docs/` — product, pack, sync, and data model specifications

## Main product goal

Build a clean web app that helps a beginner move from:

- not knowing where to start
- not being comfortable with kana
- not having a simple progression path

into:

- confident Pre-N5 completion
- structured movement into N5
- visible progress and repeatable review

## Immediate priorities

1. wire the web app to the Go backend
2. expose pack content through stable backend APIs
3. turn `pack-pre-n5-v1` into a real beginner experience
4. make dashboard, kana, quiz, and progress pages use real data

## Deprioritized for now

These are not deleted from the repository, but they are not the current project focus:

- e-ink hardware selection
- device firmware design
- offline device UI as the main product direction

Those ideas remain as future exploration only.
